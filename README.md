# ArithCV
ArithCV is a calculator project that supports both basic arithmetic and scientific calculations, demonstrating core programming concepts and mathematical operations in a simple and beginner-friendly way.

🚀 Usage
1. Run the Application

Start the calculator by running the following command:

python main.py

A window named ArithCV will open displaying your webcam feed along with the virtual calculator interface.

🖐 Hand Gesture Controls

ArithCV allows you to interact with the calculator using hand gestures.

Move your index finger to hover over calculator buttons.

Pinch your thumb and index finger together to select or press a button.

This gesture acts like a mouse click.

🔢 Normal Calculator Mode

The default mode supports basic arithmetic operations:

Button	Function
0 – 9	Enter numbers
+ - * /	Arithmetic operators
.	Decimal input
=	Calculate result
C	Clear entire equation
del	Delete last input

Example:

2 + 3 =

The calculated result will appear on the display.

🔬 Scientific Calculator Mode

Select SCI to switch to Scientific Mode.

Available functions include:

sin() – Sine

cos() – Cosine

tan() – Tangent

sqrt() – Square root

log() – Log base 10

ln() – Natural logarithm

pi – π constant

e – Euler’s number

^ – Power

rad() – Convert degrees to radians

Example calculations:

sqrt(16)
sin(rad(30))
log(100)
🔁 Switch Back to Normal Mode

Press NUM to return to the standard calculator layout.

❌ Exit the Application

Press q on your keyboard to close the program.
